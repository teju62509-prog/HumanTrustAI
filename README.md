# HumanTrustAI

## Problem Statement
In today's digital world, fake profiles and misleading online identities are increasing on social media platforms. It is difficult for users to determine whether a profile is genuine or suspicious.

## Solution
HumanTrustAI analyzes profile information such as profile link and follower count to generate a Trust Score. This helps users evaluate the reliability of an online profile quickly.

## Features
- Enter social media profile link
- Enter follower count
- AI-based Trust Score generation
- Simple and user-friendly interface

## Technology Stack
- Python
- Django
- HTML
- CSS
- JavaScript

## How to Run the Project

1. Clone the repository
2. Navigate to the project folder
3. Run the following command:

python manage.py runserver

4. Open the browser and go to:

http://127.0.0.1:8000/
